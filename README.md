# WWE

<a href="">DEMO</a>

🏆 IRON REIGN WRESTLING – Royal Rumble Arena

An interactive WWE-inspired wrestling simulation website built with HTML, CSS, and JavaScript. Create dream matches, simulate Royal Rumbles, explore wrestler profiles, and experience a cinematic wrestling arena with dynamic animations, sound effects, fireworks, and champion celebrations.

🚀 Features
🎯 Interactive Match Maker
👊 Singles Match Simulation
👑 Royal Rumble Mode
🏟️ Cinematic Wrestling Arena
🔥 Dynamic Winner Celebration
🎆 Fireworks & Confetti Effects
🔊 Arena Sound Effects
📱 Fully Responsive Design
⚡ Smooth Animations & Transitions
🎨 Modern WWE-Inspired UI

🎮 Gameplay
Singles Match
Select two wrestlers
Compare their stats
Ring the bell
Watch the battle simulation
Randomized winner selection
Royal Rumble
All wrestlers enter automatically
One superstar survives
Championship celebration sequence
